# 8INF872-PRJ02 - ARace

Dans le cadre du cours “***8INF872 - Programmation de plateformes mobiles***”, les étudiants sont amenés lors de ce 2ème TP d’explorer 4 thématiques :

- Développement d’une application AR
- Connectivité entre plusieurs appareils
- Déploiement sur les magasins d’applications en ligne
- Monétisation

L’objectif de ce projet est de développer une application mobile de type jeu en Réalité Augmentée (AR) intégrant les thématiques citées ci-dessus.

## Présentation du projet

**ARace** est un jeu de course en Réalité Augmentée où les joueurs peuvent s'affronter dans des courses en utilisant leurs appareils mobiles.

## Settings

- Version Unity : **Unity 6.2 (6000.2.1211)**
- Modules Unity : **Android Build Support**, **iOS Build Support**
